# v2.2.0 Release Notes

**Released:** 2026-03-19
**Previous release:** v2.1.0 (`1aa53ec`)

---

## Highlights

- **Full A1→C1 content for all 5 languages** — grammar, vocab, verbs, units, reading, listening, and culture now complete at every CEFR level
- **Culture module** — dedicated culture episodes with comprehension questions, vocab highlights, Did You Know notes, and YouTube video links
- **Progress export** — download a JSON backup of your learning data from the Profile page

- **A1 grammar completeness** — all 5 languages now have full A1 grammar coverage, closing gaps that existed since the modular restructure

## Content

### Culture episodes (new module)
- `CultureEpisode` type introduced: title, simplified and full body text, vocab highlights, comprehension questions, Did You Know, and optional YouTube video
- Culture episodes added for **all 5 languages at A1, A2, B1, and C1**
- YouTube IDs wired to real videos
- A1 culture fixed across all languages: vocab entries now match words from the simplified reading text; reflection/comparison questions use English prompts at A1

### A1 grammar — gaps filled (all 5 languages)

| Korean | Demonstratives (이/그/저), future tense, desire expression (고 싶다), adverbs, discourse connectors |

### Level-test questions
Added 20 questions per language at each of A2, B1, B2, and C1 (400 new questions total), bringing the test bank to a deeper pool so learners see fewer repeats on retakes.

### Japanese A1 hiragana compliance
- ~95 vocab `word` fields converted from kanji to hiragana across all categories (numbers, family, food, body, colours, clothing, places, adjectives, seasons)
- Kanji-retained words annotated with reading in Japanese parentheses: e.g. `学生（がくせい）`, `月曜日（げつようび）`
- 15 dedicated Kanji category vocab entries added (IDs 156–170) — one per essential kanji — so the U18 Beginner Kanji unit has explicit vocab cards for all 15 characters
- Grammar example sentences (`native` fields) converted to hiragana/katakana across all A1 grammar lessons

## Features

### Progress export (Profile page)
A new **"Export progress (JSON)"** button downloads a timestamped backup file containing your SRS card state, study stats, and level progress. Intended for safekeeping before clearing browser data or switching devices.

An amber banner also now explains that all data is stored only on the current device.

### Irregular verb highlighting
Irregular verb conjugation forms are now **bold** in grammar lesson examples so they stand out from the regular pattern.

### TTS-safe grammar examples
Grammar examples with non-speakable `native` text (e.g. template strings like `Subject は Object を Verb ます`) now carry a `speakText` override so the TTS button either speaks a correct alternative or stays silent, instead of reading out structural notation aloud.

## Performance

### Lazy language loading
Language data is now loaded on demand rather than bundled upfront. The initial page load drops from ~2.3 MB to ~334 KB (gzipped: ~93 KB). Each language chunk (~390 KB) loads only when that language is first selected.

| Chunk | Gzipped |
|---|---|
| Main bundle | ~93 KB |
| Spanish | ~115 KB |
| French | ~118 KB |
| Italian | ~116 KB |
| Japanese | ~146 KB |
| Korean | ~138 KB |

## UI / UX

### Page redesigns (colour system)
All major pages have been updated to use a consistent colour-coded design system — each language has an accent colour carried through headers, badges, and cards:

- **Dashboard** — tabbed layout (Path / Study / Practice / Stats) with per-section colour accents; language hero banner

Layout, font sizes, padding, and card widths improved across all pages for small screen widths.

## Infrastructure

### @myorg packages wired
The app now consumes `@myorg/quiz-engine` (drill hook + done screen), `@myorg/tts` (speech engine + SpeakButton), and `@myorg/srs` (SM-2 algorithm) from the monorepo, rather than maintaining those as inline app code.

### Netlify SPA redirect
`public/_redirects` added so direct URL access and hard refreshes work correctly on Netlify (`/* /index.html 200`).
