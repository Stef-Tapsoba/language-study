## v2.0.0 — March 2026

### New features

**Spaced repetition flashcards**
SM-2 scheduling for all vocabulary. Cards are rated *Got it* / *Not yet* and resurface at optimal intervals. 10 new cards per day per language; due-count badge on the dashboard. 3D flip animation with TTS auto-play on card appear and flip.

**Text-to-speech**
Web Speech API pronunciation throughout: speaker buttons on vocabulary, verb conjugation tables, flashcards, reading passages, and listening exercises. Slow/normal rate toggle on the listening player.

**Reading comprehension (CE)**
Graded passages with inline vocabulary glosses and multiple-choice comprehension questions. Available at A1 and A2 for all five languages.

**Listening comprehension (CO)**
TTS-driven audio exercises with transcript and comprehension questions. Available at A1 and A2 for all five languages.

**Study statistics**
- Global streak chip (🔥 Nd) in the navigation bar — any language studied counts
- 14-day review bar chart on the Dashboard Stats tab, per language
- Accuracy percentage over the last 14 days
- Streak and total items learned shown on the Profile page

**B1 content**
Intermediate-level grammar, vocabulary, verbs, units, placement questions, and level-test questions added for all five languages.

### Improvements

- Dashboard expanded to five tabs: Path / Study / Practice / Test / **Stats**
- Language reset now cleanly removes flashcard state and review history for that language
- All source files carry a standard path + description header comment
